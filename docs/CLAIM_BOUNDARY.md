# Claim Boundary

## Supported claim

OpenLine Receipt Gate v0.1.1 shows that a risky agent boundary crossing can be wrapped with a tiny Python context manager that emits hash-chained receipts and fails closed when required proof is missing.

## Not claimed

This repo does not claim:

- full agent safety;
- full benchmark auditing;
- full memory integrity;
- public-key signatures;
- production-grade access control;
- replacement for observability platforms;
- reversal of side effects after execution.

It is a developer primitive.
